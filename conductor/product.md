# Initial Concept
The goal is to showcase how to build a scalable and flexible data platform that combines the best aspects of data lakes and data warehouses, enabling advanced analytics, machine learning, and business intelligence, combining familiar and portable open source technologies together with cloud native capabilities and level of management.

# Product Guide - Open Data Lakehouse Demo

## Vision
The Open Data Lakehouse Demo is a comprehensive project designed to showcase a modern, flexible, and scalable data architecture. By integrating open-source technologies like Spark and Kafka with cloud-native capabilities like BigQuery, it demonstrates a portable data platform that breaks down silos and enables advanced analytics across different environments.

## Target Audience
- **Data Engineers/Architects:** Professionals responsible for data architecture and platform design.
- **Solution Architects/DevOps:** Teams looking for end-to-end cloud data demos.
- **Data Analysts/Scientists:** Users interested in querying and visualizing lakehouse data.

## Core Goals
- **Showcase Open Formats:** Highlight the portability and interoperability of data using open formats (like Parquet and Iceberg).
- **End-to-End Data Pipeline:** Demonstrate a complete data journey from ingestion to visualization.
- **Iceberg Ecosystem Mastery:** Show how to effectively use the Iceberg catalog to manage tables consistently across Spark and BigQuery.

## Key Features
- **Streaming Ingestion:** Real-time data ingestion and processing using Spark and Kafka.
- **Iceberg Catalog Integration:** Seamless management of table metadata using a unified Iceberg catalog, enabling interoperability between Spark and BigQuery.
- **Interactive Dashboard:** A web-based dashboard (Next.js/Flask) for monitoring bus lines and visualizing system status.
- **Open Data Lakehouse:** Deep integration of open-source engines (Spark) with enterprise data warehouses (BigQuery).
- **Infrastructure as Code:** Fully automated deployment using Terraform.

## Success Criteria
- **Educational Value:** Users gain a clear, actionable understanding of open data lakehouse principles, specifically focusing on the power of the Iceberg format and catalog for cross-platform data management.
